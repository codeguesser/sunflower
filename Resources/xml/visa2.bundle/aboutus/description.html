關於我們
  • 公司簡介 新華旅遊始創於1974年，為香港少數獨立經營的知名老牌旅行社，憑著進取熱誠的幹勁與不斷尋求新突破的宗旨，在旅遊界建立深厚根基，質素有口皆碑。目前總行及12間分社遍佈港、九、新界。服務範疇包括組辦外遊旅行團往世界各地(遍及中國大陸、全亞洲、歐美、地中海、中東、澳洲、東南非、南美大部分國家)、自助旅遊套票、公司及團體獨立組團、全球機票、酒店預訂、火車證及郵輪假期預訂、代辦世界各國簽證、中國公民出境遊等等。
  • 新華旅遊宗旨
    • 服務範疇 應有盡有
      新華旅遊素以高質素旅遊及票務服務見稱。旅遊部除辦理前往世各地之長短線團隊； 更推出開心自助逍遙遊，讓旅客隨喜好調配時間安排愜意節目；並有專為公司團體或具有特殊旅遊目的人士而度身設計的行程。 新華旅遊的票務部提供世界各地航空公司之機票訂位及開票服務，成積斐然，為本港著名大企業所樂用的旅行社。 此外，本公司更有專人為顧客辦理簽證，代訂各國酒店房間等，效率一流。
    • 提昇質素 細意週到
      新華旅遊時刻以提昇旅遊品質及服務為宗旨，為顧客提供超值及最佳享受，令每個團友都有一個開心難忘的假期。 訓練有素，慇懃熱誠的員工事事以客為尊。新華並率先採用全電腦操作系統，為其專業服務效率更添實力後盾。 本公司更積極在港九各地增設更多分社，以應付日益增長的需求，方便不同區域的客戶。各款新華旅行團的行程均緊湊充實，將最具遊覽價值的名勝重點，全部展現閣下眼前。 沿途由專業領隊帶領，經驗導遊講解風光趣聞，令閣下充份領略賓至如歸的旅遊樂趣。
    • 創新氣象 開心至上
      新華旅遊精益求精，用象徵活力朝氣的向日葵為標誌，以無比信心邁向另一個新高峰，同時亦希望為團友帶來向日葵一樣燦爛歡欣的旅遊體驗。
